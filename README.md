# convert_yt_to_mp3_mp4
Script convert yt to mp3 dan mp4 menggunakan python3, bahan bahan

Termux:<br>
pkg update && pkg upgrade && pkg install git && pkg install python3 && pip3 install pytube && pip3 install tqdm<br>
git clone https://github.com/xxoprt/convert_yt_to_mp3_mp4 && cd convert_yt_to_mp3_mp4 && python3 convert.py<br>
kalian input mau mp3 atau mp4, dan jika sudah di pilih kalian tinggal masukkan url, dan tunggu<br>

Ubuntu/Kali/Parrot :<br>
sudo apt update && sudo apt install git python3 python3-pip && pip3 install pytube tqdm && git clone https://github.com/xxoprt/convert_yt_to_mp3_mp4 && cd convert_yt_to_mp3_mp4 && python3 convert.py<br>
